# 英语学习计划（每日2小时版）

## 📌 总目标
1. 实现日常沟通无障碍
2. 无字幕观看英美影视剧
3. 流畅阅读计算机领域文献
4. 快速理解英语新闻
5. 考研英语达到75+水平

## 📅 阶段划分（6个月周期）
```plaintext
▶ 基础巩固期（第1-2月）：词汇语法筑基 + 听力口语启蒙
▶ 专项突破期（第3-4月）：四维能力强化 + 专业领域渗透
▶ 考研冲刺期（第5-6月）：应试技巧打磨 + 全真模拟训练
```

## 📖 每日学习框架（120分钟）

### 🌅 晨间模块（30分钟）

```
1. 词汇攻坚（15min）
   - 考研高频词30个（《考研词汇闪过》）
   - 计算机术语10个（IEEE论文/技术文档）
2. 新闻速读（15min）
   - 选择BBC Technology板块报道
   - 完成"3R阅读法"：Read-Recite-Review
```

### 🎧 午间模块（30分钟）

```
1. 听力精炼（20min）
   - 材料选择：TED-ED技术类视频/Cambridge IELTS听力
   - 训练方法：三遍精听法（盲听→跟写→对照）
2. 影子跟读（10min）
   - 使用《EnglishPod》商务对话
   - 同步模仿语音语调
```

### 🌃 晚间模块（60分钟）

```
1. 专业文献研读（30min）
   - 精读ACM/IEEE论文1-2页
   - 制作术语库+长难句分析
2. 影视输入（15min）
   - 观看《硅谷》等科技类美剧
   - 开启英文字幕做场景复述
3. 写作输出（15min）
   - 交替进行：考研作文/技术文档摘要
   - 使用Grammarly进行语法修正
```

## 🔄 周末强化

```
✓ 周六：全真模考（近5年考研真题）
✓ 周日：主题拓展（MOOC技术课程+技术博客翻译）
```

## 📚 推荐资源矩阵

| 类型 | 推荐内容                                     |
| :--- | :------------------------------------------- |
| 词汇 | 《考研词汇闪过》+《Cambridge IT Dictionary》 |
| 听力 | TED-ED技术专题+《Scientific American》播客   |
| 阅读 | IEEE Xplore论文+《经济学人》科技版块         |
| 口语 | Cambly技术专场+《English for Tech》课程      |
| 写作 | 《顾家北手把手》+ Overleaf学术模板           |

## ⚠️ 关键执行原则

1. 词汇联想记忆：用Notion建立「技术词场」数据库
2. 听力场景迁移：将美剧台词转化为日常对话素材
3. 阅读双向强化：技术文档与考研阅读交叉训练
4. 输出倒逼输入：每周录制1次技术概念英文解说

## 📈 进度监控

```
+ 每周进行CEFR自测（EF SET官网）
+ 每月更新技术文献阅读速度记录
! 建立错题博物馆：分类整理听力/阅读高频错误
```

> 注：本计划需配合「刻意中断」机制——每学习45分钟强制休息5分钟，采用番茄工作法提升专注度。建议安装RescueTime进行学习效率监控。
